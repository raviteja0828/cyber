DoS Attack

Steps to Secure:
  1. Immediately disconnect from the internet
  2. Contact your ISP to report the attack

{% elif prediction_entry.prediction_result == 'normal' %}

Normal Activity

Maintain Security:
  • Keep all systems and software up-to-date
  • Use strong, unique passwords
  • Enable two-factor authentication
  • Regularly back up your data
  • Educate users about security best practices
{% elif prediction_entry.prediction_result == 'probe' %}

Probe Attack

Steps to Secure:
  1. Review and strengthen firewall rules
  2. Disable unnecessary services and ports
  3. Implement intrusion detection systems (IDS)
  4. Use port knocking or other obfuscation techniques
  5. Regularly scan for vulnerabilities

Important: Monitor logs for unusual scanning activity and consider using honeypots to detect and divert probing attempts.

{% elif prediction_entry.prediction_result == 'r2l' %}

R2L Attack

Steps to Secure:
  1. Immediately change all passwords
  2. Implement strict access controls
  3. Enable and review system logs
  4. Use strong encryption for remote access
  5. Implement network segmentation

Important: Conduct a thorough security audit and consider implementing multi-factor authentication for all remote access.

{% elif prediction_entry.prediction_result == 'u2r' %}

U2R Attack

Steps to Secure:
  1. Immediately isolate the affected system
  2. Revoke and reissue all user credentials
  3. Apply all security patches and updates
  4. Implement principle of least privilege
  5. Use application whitelisting

Important: Conduct a full system audit, including all installed software and user accounts. Consider implementing a robust endpoint detection and response (EDR) solution.
